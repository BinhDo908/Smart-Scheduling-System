# Smart Teacher Scheduling System (VJU Edition)

Hệ thống xếp thời khóa biểu thông minh cho môi trường đa ngành và đa cơ sở.

## 🌟 Tính năng nổi bật
- **Auto-Staffing**: Tự động gán giảng viên theo Major và Chuyên môn.
- **Campus-Aware**: Tự động phân bổ sinh viên Y1 tại Hòa Lạc, Y2-Y4 tại Mỹ Đình.
- **Parallel Computing**: Sử dụng đa luồng để tối ưu hóa tốc độ tìm kiếm nghiệm.
- **Priority Greedy**: Ưu tiên xếp các môn Bắt buộc (Core) trước môn Tự chọn (Elective).

## 🛠 Tech Stack
- Python 3.11+
- Pandas (Xử lý dữ liệu)
- Concurrent.futures (Tính toán song song)
